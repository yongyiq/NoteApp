# 欢迎使用 NoteFlow

NoteFlow 是一款支持 **Markdown 渲染**、**PDF 查看**和**图片预览**的智能笔记软件。

## 功能特色

| 功能 | 说明 |
|------|------|
| Markdown 编辑 | 实时编辑，支持语法高亮 |
| 分屏预览 | 左侧编辑，右侧实时渲染 |
| PDF 查看 | 内置 PDF 阅读器，支持缩放翻页 |
| 图片预览 | 支持 PNG / JPG / GIF / WebP / SVG |
| 深色模式 | 一键切换，护眼阅读 |
| 文件夹管理 | 笔记分组整理 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存笔记 |
| `Ctrl+B` | 粗体 |
| `Ctrl+I` | 斜体 |
| `Ctrl+/` | 切换预览 |
| `Ctrl+D` | 删除当前行 |
